What happens when you initialize a repository? Why do you need to do it?

	A .git folder is created which contains all the metadata needed for git.

How is the staging area different from the working directory and the repository?
What value do you think it offers?
	
	The staging area contains only the files that you have chosen for the
	upcoming commit. The working directory and repository contain all the
	files.

How can you use the staging area to make sure you have one commit per logical
change?

	Only add files to the staging area that are part of a single logical change.

What are some situations when branches would be helpful in keeping your history
organized? How would branches help?

	When experimenting with different features and taking the program/game
	in different directions, such as different languages.

How do the diagrams help you visualize the branch structure?

	Diagrams help us to understand the flow of the branches and work only on
	the branch that we intend to be working on.
